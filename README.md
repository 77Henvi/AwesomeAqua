# 🐠 AwesomeAqua

> Aquarium fish marketplace — simple, direct, real

---

##  Overview

A Fullstack Fish Shop web platform for buying and selling aquarium fish
connected directly through LINE

Auth → Supabase
DB → Supabase
Storage → Supabase
Frontend → Vanilla JS
Admin → CRUD + Dashboard

---

## ⚙️ Features

* Browse available fish
* View basic details
* Contact seller via LINE instantly
* Admin Interface , Manage and Correct fishs data Real-Time with SupaBase DB
